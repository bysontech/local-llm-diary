# 実装サイクル計画（Cycle 3 / 1〜2日）

## 期間
- （例）2026/02/09 〜 2026/02/10

## 今回のユースケース（縦切り）
1. PWA更新体験
- アプリを通常起動
- 新版があれば「更新しますか？」が出る
- ユーザーが更新を選ぶまで勝手に更新されない
- 更新後もIndexedDBデータが残る

2. エクスポート/インポート
- 設定画面からエクスポート → JSONファイルが保存できる
- 全データ削除 → 一覧が空
- インポート → entries が復元される
- 設定（要約ON/OFF）も復元される

## ゴール
- PWAが「事故らず使える」状態
- データが消える不安を最小化（JSONバックアップ）

## やらないこと
- 暗号化
- クラウド同期
- WebGPU/モデルDL
- 高度検索（転置インデックス等）

## 人が判断するポイント
- インポートは「上書き」か「追記」かを決める（MVPは上書き推奨）
- 本文をログに出さない
- 依存追加は最小
