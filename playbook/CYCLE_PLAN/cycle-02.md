# 実装サイクル計画（1〜2日）

## サイクル期間
- （例）2026/02/08 〜 2026/02/09

## 今回のユースケース（縦切り）
- 詳細 → 編集 → 保存（要約再生成） → 詳細で反映確認
- 詳細 → 削除 → 一覧から消える
- 設定 → 要約ON/OFF → 保存挙動が変わる
- 設定 → 全データ削除 → 一覧が空になる

## ゴール
- Edit/Delete/Settings最小が動き、日記として“最低限使える”

## 担当AI
- Claude Code：編集・削除・設定・DB関数追加
- Mac版Codex：画面UIの配線、ボタン追加、表示整形
- Cursor：動作確認しながら微修正、UX調整
